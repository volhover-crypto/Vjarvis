# Бионическая стойка для крепления датчиков

## Параметры

| Параметр | Значение |
|----------|----------|
| Общая высота | 2000 мм |
| Диаметр (верх) | 30 мм |
| Диаметр (низ) | 50 мм |
| Количество секций | 10 |
| Длина секции | 200 мм |
| Толщина стенки | 3 мм |
| Диаметр штыря | 8 мм |
| Длина штыря | 15 мм |
| Глубина гнезда | 12 мм |

## Структура

```
base_plate.stl      — Основание (плита с отверстиями для крепления к земле)
section_01.stl      — Секция 1 (нижняя, диаметр 50→48 мм)
section_02.stl      — Секция 2 (диаметр 48→46 мм)
section_03.stl      — Секция 3 (диаметр 46→44 мм)
section_04.stl      — Секция 4 (диаметр 44→42 мм)
section_05.stl      — Секция 5 (диаметр 42→40 мм)
section_06.stl      — Секция 6 (диаметр 40→38 мм)
section_07.stl      — Секция 7 (диаметр 38→36 мм)
section_08.stl      — Секция 8 (диаметр 36→34 мм)
section_09.stl      — Секция 9 (диаметр 34→32 мм)
section_10.stl      — Секция 10 (верхняя, диаметр 32→30 мм)
top_cap.stl         — Верхняя крышка (крепление датчика)
```

## Сборка

1. Напечатайте все детали на 3D-принтере (PETG или ASA)
2. Начните с основания — закрепите болтами М8
3. Вставьте штырь секции 1 в гнездо основания
4. Последовательно наращивайте секции 2→10
5. Установите верхнюю крышку
6. Закрепите датчик на крышке (отверстия М5)

## Рекомендации по печати

- **Материал:** PETG (устойчивость к погоде) или ASA (UV-устойчивость)
- **Заполнение:** 100% для несущих элементов
- **Ориентация:** вертикально (вдоль оси Z)
- **Поддержки:** минимальные (только для гнёзд)
- **Слой:** 0.2 мм (баланс скорости и качества)

## Генерация

```bash
python3 generate_stand_simple.py
```
