# Бионическая стойка для датчиков — версия с ферменной структурой

## Параметры

| Параметр | Значение |
|----------|----------|
| Общая высота | 2000 мм |
| Диаметр (верх) | 30 мм |
| Диаметр (низ) | 50 мм |
| Количество секций | 10 |
| Длина секции | 200 мм |
| Толщина стенки | 3 мм |
| Диаметр штыря | 8 мм |
| Длина штыря | 15 мм |
| Глубина гнезда | 12 мм |
| Стрингеров на секцию | 3-6 (адаптивно) |
| Колец жёсткости | 1-4 (адаптивно) |

## Структура

Каждая секция содержит:
- **Внешняя оболочка** — усечённый конус с толщиной стенки 3 мм
- **Внутренняя ферма** — стрингеры (вертикальные рёбра) + диагональные распорки + горизонтальные кольца
- **Соединения** — штырь-гнездо для стыковки секций

Адаптивная ферма: на нижних (толстых) секциях — 6 стрингеров + диагонали + 4 кольца, на верхних (тонких) — 3 стрингера + 2 кольца.

## Сборка

1. Напечатайте все детали (PETG или ASA)
2. Закрепите основание болтами М8
4. Вставьте штырь секции 1 в гнездо основания
5. Последовательно наращивайте секции 2→10
6. Установите верхнюю крышку
7. Закрепите датчик (отверстия М5)

## Рекомендации по печати

- **Материал:** PETG (устойчивость к погоде) или ASA (UV-устойчивость)
- **Заполнение:** 100% (ферма уже внутри)
- **Ориентация:** вертикально (вдоль оси Z)
- **Поддержки:** минимальные (только для гнёзд)
- **Слой:** 0.2 мм
