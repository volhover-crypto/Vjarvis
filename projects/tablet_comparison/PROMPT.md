# Промпт: глубокое сравнение планшетов на российских маркетплейсах

## Промпт для нейросети

```
Роль: Ты — аналитик потребительской электроники с опытом работы на российском рынке. Ты знаешь специфику маркетплейсов, реальные цены, наличие и сроки доставки.

Задача: Проведи глубокое сравнение текущих предложений планшетов на российских маркетплейсах (Ozon, Wildberries, Яндекс Маркет, М.Видео, Ситилинк, DNS).

Перед началом анализа задай уточняющие вопросы (максимум 3):
1. Бюджет (диапазон в рублях)?
2. Основное назначение (работа, учёба, развлечения, рисование)?
3. Предпочтительный размер экрана или ОС (Android, iPadOS, Windows)?

После получения ответов проведи анализ по следующей структуре:

## 1. Методология
- Дата анализа
- Проверенные маркетплейсы
- Примечания по недоступности данных

## 2. Сравнительная таблица (топ-5–7 моделей)
Для каждой модели:
- Модель, бренд
- Экран: размер, тип матрицы, разрешение
- Процессор, RAM, накопитель
- ОС и версия
- Батарея (ёмкость, время работы по отзывам)
- Цена: минимальная найденная + на каком маркетплейсе
- Рейтинг и количество отзывов

## 3. Детальный разбор каждой модели
Для каждой модели из таблицы:
- Сильные стороны (3–5 пунктов)
- Слабые стороны (3–5 пунктов)
- Типичные проблемы из отзывов (цитируй характерные жалобы)
- Для каких задач подходит лучше всего
- Для каких задач НЕ подходит

## 4. Сравнение по сценариям использования
Сравни модели попарно по каждому сценарию (работа, учёба, развлечения, рисование). Укажи победителя в каждой категории и обоснуй почему.

## 5. Ценовые сегменты
Разбей найденные модели на ценовые сегменты (до 15000, 15000–30000, 30000–50000, 50000+). В каждом сегменте укажи лучший выбор и альтернативу.

## 6. Рекомендации
- Лучший планшет по соотношению цена/качество
- Лучший бюджетный вариант
- Лучший премиальный вариант
- На что обратить внимание при покупке (подделки, ревизии, комплектация)

Ограничения: Не выдумывай характеристики и цены. Если данные недоступны — честно укажи это. Объём — 2000–3000 слов.
```

## Инструкция по использованию

1. Скопируй промпт
2. Вставь в LLM (Claude, GPT, Gemini, GigaChat, YandexGPT)
3. Ответь на уточняющие вопросы нейросети
4. Получи полный анализ с таблицами и рекомендациями
