# Плановые cron-задачи (ожидают активации после восстановления gateway-связи)

## Задача 1: Ежедневная консолидация памяти
- **Расписание (cron):** `55 02 * * *` (каждый день в 02:55 UTC)
- **Тип сессии:** изолированная (`isolated`)
- **Действие:** 
  - Создать или обновить файл `memory/YYYY-MM-DD.md` (где YYYY-MM-DD — сегодняшняя дата)
  - Записать:
    - Что мы делали сегодня
    - Ключевые решения и их обоснования
    - Открытые вопросы
    - Контекст для следующей сессии
  - Обновить `MEMORY.md`, если появились важные долгосрочные инсайты
- **Флаг:** `--delete-after-run` не требуется (это повторяющаяся задача)
- **Доставка:** не объявлять в чат (выполняется в фоне)
- **Модель:** использовать текущую сессионную настройку (или `openrouter/nvidia/nemotron-3-super-120b-a12b:free`)
- **Примечание:** Требует авторизованного подключения к gateway

## Задача 2: Еженедельный дайджест памяти
- **Расписание (cron):** `0 4 * * 1` (каждый понедельник в 04:00 UTC)
- **Тип сессии:** изолированная (`isolated`)
- **Действие:**
  - Прочитать все файлы `memory/*.md` за последние 7 дней
  - Обновить раздел `[Неделя: YYYY-MM-DD -- YYYY-MM-DD]` в `MEMORY.md` с резюме
  - Удалить дублирующиеся записи в `MEMORY.md`
- **Требования:**
  - Учесть переход между месяцами/годами при выборке за 7 дней
  - Сохранять только значимую информацию — не копировать сырые логи
- **Флаг:** повторяющаяся, без удаления после запуска
- **Доставка:** фоновое выполнение, без объявления
- **Модель:** использовать стандартную настройку
- **Примечание:** Требует авторизованного подключения к gateway

## Инструкция по применению (когда будет доступен gateway):
Для каждой задачи выполнить что-то вроде:

```
openclaw cron add \
  --name "<название>" \
  --cron "<выражение>" \
  --session isolated \
  --message "<prompt>" \
  [--model <model>] \
  [--thinking <level>] \
  [--tz <timezone>] \
  [--announce|--no-announce] \
  [--channel <channel>] \
  [--to <recipient>]
```

Пример для первой задачи (без annonuce, без указания tz — по умолчанию UTC):

```
openclaw cron add \
  --name "Ежедневная консолидация памяти" \
  --cron "55 02 * * *" \
  --session isolated \
  --message "Создай или обнови файл memory/[сегодняшняя дата].md. Запиши: что мы делали сегодня, ключевые решения, открытые вопросы, контекст для следующей сессии. Обнови MEMORY.md если появились важные долгосрочные инсайты."
```

Для второй задачи (можно добавить `--tz "Europe/Moscow"` если нужно локальное время, но в расписании не указано — оставляем UTC):

```
openclaw cron add \
  --name "Еженедельный дайджест памяти" \
  --cron "0 4 * * 1" \
  --session isolated \
  --message "Сделай еженедельный дайджест: прочитай все memory/*.md за последние 7 дней, обнови MEMORY.md разделом [Неделя: дата], удали дублирующиеся записи."
```
