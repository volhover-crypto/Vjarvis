# Technical Writing — Техническое писательство

## Когда использовать
Создание технических документов: спецификации, руководства пользователя, пояснительные записки, технические задания (ТЗ), обоснования, отчёты, документация API.

## Принципы

1. **Структура прежде всего.** Каждый документ начинается с оглавления и введения с целями.
2. **Одно предложение — одна мысль.** Короткие предложения. Нет прилагательных без необходимости.
3. **Конкретика.** Числа, единицы измерения, допуски. Не «высокая точность», а «точность ±0.5%».
4. **Аудитория.** Пиши для того, кто будет читать. Инженеру — технические детали. Руководителю — выводы и цифры.
5. **Визуализация.** Таблицы для сравнений, схемы для процессов, рисунки для конструкций.

## Структура технического документа

### Пояснительная записка / Обоснование
```
1. Введение (цели, задачи)
2. Анализ текущего состояния
3. Предлагаемое решение (техническое описание)
4. Экономическое обоснование (CAPEX, OPEX, окупаемость)
5. Риски и ограничения
6. Выводы и рекомендации
7. Приложения (чертежи, таблицы, расчёты)
```

### Техническое задание (ТЗ)
```
1. Общие сведения (наименование, основание, сроки)
2. Назначение и цели
3. Требования (функциональные, технические, к документации)
4. Порядок выполнения (этапы, контрольные точки)
5. Порядок приёмки
6. Приложения
```

### Руководство пользователя
```
1. Назначение
2. Системные требования
3. Установка / Настройка
4. Описание операций (пошагово)
5. Неисправности и устранение
6. Техническое обслуживание
7. Приложения
```

## Язык и стиль

- **Запрещено:** «в настоящее время», «является», «осуществляется», «данный» (в значении «этот»), «вышеуказанный»
- **Предписано:** активный залог, конкретные глаголы, числовые значения
- **Ссылки на ГОСТ:** ГОСТ 2.105-2019 (текстовые документы), ГОСТ 19.201-78 (ТЗ), ГОСТ 34.201-89 (техническое задание)
- **Единицы:** СИ, запись через пробел (25 км, не 25км)

## Формат вывода

Markdown. Заголовки — `#`, `##`, `###`. Таблицы — GFM. Списки — нумерованные или маркированные. Длина — по запросу, не более 3000 слов на один документ без явного указания.
