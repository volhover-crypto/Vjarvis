# Папка REZERV — Резервные копии инструкций и конфигов

**Назначение:** Хранение актуальных версий инструкций, правил и описаний структуры проектов для быстрого восстановления или аудита.

## Содержимое папки:

1. **RAG_N8N_INSTRUCTION.md**
   - **Для чего:** Инструкция по работе с RAG-разделом "n8n-orchestration".
   - **Как использовать:** Описывает процесс добавления новых статей, структуру папок и запуск скриптов обработки.

2. **CORE_IDENTITY.md**
   - **Для чего:** "Ядро" личности Джарвиса. Содержит базовые доступы, правила безопасности и технические параметры сервера.
   - **Как использовать:** Точка отсчета при полном восстановлении агента после сбоя.

3. **SOUL.md**
   - **Для чего:** Принципы общения, стиль (Human-Style), проактивность и границы дозволенного.
   - **Как использовать:** Определяет "характер" агента и тон взаимодействия.

4. **EMERGENCY_RECOVERY.txt**
   - **Для чего:** Экстренный протокол восстановления.
   - **Как использовать:** Отправить этот текст агенту в чат, если он "забыл" контекст или перестал адекватно отвечать.

5. **SESSION_BOOTSTRAP.md**
   - **Для чего:** Стартовый пакет для инициализации новой сессии.
   - **Как использовать:** Содержит список_skills, пути к файлам и порядок загрузки контекста при старте.
