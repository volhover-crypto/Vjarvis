# memory/principles_lib.md — Библиотека стратегических принципов (EvolveR)

> Хранит абстрактные стратегические принципы, извлечённые из опыта агента через процесс offline self-distillation (EvolveR).  
> Каждый принцип имеет:
> - Текст принципа
> - Метрика эффективности (applied_count, success_rate, last_applied)
> - Источник (из какой записи/опыта получен)
> - Статус (active, deprecated, under_review)
>  
> Формат основан на EvolveR:  
> - Семантическая дедупликация предотвращает взрывной рост дублирующих принципов  
> - Интеграция позволяет новым принципам обновлять или заменять существующие  
> - Метрика эффективности ранжирует принципы по историческому успеху  
>  
> Принципы с success_rate < 0.4 и applied_count > 5 → предлагать пересмотр  
>  
> Используется в фазе Online Interaction: агент может вызвать `<search_experience>` для получения релевантных принципов перед рассуждением
