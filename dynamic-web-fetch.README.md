# Навык dynamic-web-fetch

Успешно создан навык для извлечения содержимого с динамических веб-страниц.

## Что создано

- **Навык**: `dynamic-web-fetch.skill` - готовый к использованию пакет навыка
- **Исходный код**: `/root/.openclaw/workspace/skills/dynamic-web-fetch/dynamic-web-fetch/`
- **Документация**: Полная в SKILL.md и references/

## Компоненты навыка

### Основной скрипт
`scripts/fetch-dynamic-content.py` - Python скрипт, который:
- Принимает URL и опции ожидания контента
- Использует Node.js версию Playwright через дочерний процесс
- Ждет загрузки контента (селектор или networkidle)
- Извлекает содержимое в формате text, markdown или HTML
- Возвращает структурированный результат

### Справочные материалы
В директории `references/`:
- `wait-selector.md` - Руководство по ожиданию селекторов
- `network-idle.md` - Руководство по ожиданию сетевого бездействия  
- `table-extraction.md` - Методы извлечения табличных данных
- `examples.md` - Практические примеры использования

## Возможности

Навык позволяет:
- Загружать JavaScript-тяжелые страницы и SPA
- Ждать появленияSpecific элементов в DOM
- Ждать сетевого бездействия для AJAX-контента
- Извлекать текст, таблицы и HTML-содержимое
- Преобразовывать результаты в markdown для дальнейшего анализа
- Обрабатывать ошибки и таймауты

## Использование

Навык можно вызвать через команду:
```bash
python3 fetch-dynamic-content.py <URL> [опции]
```

Опции включают:
- `--wait-for <selector>` - CSS селектор или 'networkidle' для ожидания
- `--wait-timeout <ms>` - Таймаут ожидания (по умолчанию 30000)
- `--extract-mode <mode>` - Режим извлечения: text, markdown, html
- `--selector-type <type>` - Тип селектора: css или xpath
- `--custom-script <js>` - Пользовательский JavaScript для выполнения
- `--output-format <fmt>` - Формат вывода: json или plain

## Применение

Навык особенно полезен для:
- Сбора данных с динамических веб-сайтов
- Извлечения информации из веб-приложений
- Парсинга контента, требующего JavaScript для отображения
- Автоматизации сбора данных для анализа и исследований
