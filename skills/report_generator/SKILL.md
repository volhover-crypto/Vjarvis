# Report Generator — Генератор отчётов

## Когда использовать
Запросы на формирование отчётов, сводок, дашбордов по данным мониторинга, агрономическим наблюдениям, результатам анализов.

## Типы отчётов

### 1. Суточная сводка метео
- Мин/макс/средняя температура
- Сумма осадков
- Средняя влажность
- Скорость ветра
- GDD за день (если есть данные)

### 2. Недельный отчёт по винограднику
- Фенофаза (BBCH)
- Накопленный GDD
- Состояние почвы (влажность, EC, pH)
- Риск болезней (по погодным условиям)
- Рекомендации по поливу/защите

### 3. Отчёт по сенсорам
- Статус всех датчиков (online/offline)
- Временные ряды показаний
- Инциденты (выход за пороги, зависания)
- Качество данных (% пропусков)

### 4. Сезонный отчёт
- Суммарный GDD за сезон
- Динамика NDVI
- Урожайность по участкам
- Сравнение с прошлыми сезонами

## Формат вывода

### Telegram (основной)
- Краткий, структурированный
- Эмодзи для быстрого сканирования
- Ключевые цифры выделены
- Рекомендации в конце

### Markdown (для Vault)
- Полный формат с таблицами
- Ссылки на источники данных
- Графики (если сгенерированы)

### PDF (для архива)
- Через Python (matplotlib + reportlab)
- Для официальных отчётов

## Шаблон суточной сводки
```
🌤 Сводка за [дата]

🌡 Температура: [min]°C / [avg]°C / [max]°C
💧 Влажность: [min]% / [avg]% / [max]%
🌧 Оадки: [sum] мм
💨 Ветер: [avg] м/с, [direction]

🍇 Виноградник:
- Фенофаза: BBCH [code]
- GDD сегодня: [value] (всего: [total])
- Влажность почвы: [value]% ([status])

⚠️ Алерты: [список или "нет"]
📋 Рекомендации: [список]
```
