# Harness Engineering Overview

Harness engineering focuses on building a deterministic "harness" around AI agents to improve reliability.

Key concepts:
- Structural tests: verify the agent's environment and constraints.
- Linters: enforce coding and configuration standards.
- Architectural limits: define boundaries that the agent should not exceed.
