# 🚀 Инструкция: Начать новую сессию

**Команда:** `new` или `/new`

---

## Шаг 1: Загрузить SESSION_BOOTSTRAP.md

При старте новой сессии AI-агент **ДОЛЖЕН** выполнить:

```
1. Загрузить файл: /root/.openclaw/workspace/SESSION_BOOTSTRAP.md
   ИЛИ
   Скачать с Google Drive: istohniki/Инструкции/SESSION_BOOTSTRAP.md
```

## Шаг 2: Прочитать по приоритету

**Priority 1:** Identity & Soul (кто я, как себя вести)  
**Priority 2:** Tools & Access (какие есть доступы)  
**Priority 3:** RAG Structure (где база знаний)  
**Priority 4:** Skills (какие навыки установлены)

## Шаг 3: Проверить контекст

```bash
# Проверить последние memory файлы
ls -lt ~/.openclaw/workspace/memory/ | head -5

# Проверить HEARTBEAT.md на задачи
cat ~/.openclaw/workspace/HEARTBEAT.md

# Проверить Google Drive доступ
python3 -c "from google.oauth2.credentials import Credentials; print('✅ Google Drive OK')"
```

## Шаг 4: Сообщить о готовности

```

📚 RAG-база: 67 файлов по 5 темам
```

## Автоматизация (опционально)

Можно создать скрипт `session_init.py`, который будет:
1. Загружать SESSION_BOOTSTRAP.md
2. Парсить ключевую информацию
3. Выводить сводку для агента

---

**Важно:** SESSION_BOOTSTRAP.md нужно обновлять при:
- Добавлении новых навыков
- Изменении API ключей
- Расширении RAG-базы
- Изменении конфигурации

**Команда обновления:**
```bash
python3 ~/.openclaw/workspace/scripts/generate_bootstrap.py
```

(Скрипт генерации создаётся отдельно)
