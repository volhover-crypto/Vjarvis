## Что включает навык

Навык позволяет извлекать содержимое с динамических веб-страниц используя headless-браузер Playwright с następuющими возможностями:

### Основные функции
- Принимает URL целевой страницы для загрузки
- Использует headless-браузер Playwright через Node.js для обхода ограничений среды
- Ожидает загрузки контента через:
  - CSS селекторы (например, ".sorts-list", "#content")
  - XPath селекторы
  - Сетевое бездействие (networkidle) для AJAX-контента
- Извлекает содержимое в различных форматах:
  - **text** - чистый текст содержимого страницы
  - **markdown** - преобразованный в markdown для удобного анализа
  - **html** - исходный HTML содержимое
- Поддерживает выполнение пользовательского JavaScript после загрузки страницы
- Возвращает структурированный результат с метаданными (заголовок, финальный URL, статус код и т.д.)

### Справочные материалы
Включает подробные руководства по:
- Ожиданию селекторов (CSS/XPath)
- Ожиданию сетевого бездействия для динамического контента
- Извлечению и обработке табличных данных
- Практическим примерам использования для различных сценариев

## Как использовать

Навык можно вызвать через команду:
```bash
python3 fetch-dynamic-content.py <URL> [опции]
```

### Основные опции
- `--wait-for <selector>` - CSS селектор для ожидания элемента или 'networkidle'
- `--wait-timeout <ms>` - Таймаут ожидания (по умолчанию 30000 мс)
- `--extract-mode <mode>` - Режим извлечения: text, markdown, html (по умолчанию text)
- `--selector-type <type>` - Тип селектора: css или xpath (по умолчанию css)
- `--custom-script <js>` - Пользовательский JavaScript для выполнения после загрузки
- `--output-format <fmt>` - Формат вывода: json или plain (по умолчанию json)

### Пример использования
Для извлечения списка сортов с винодельческого сайта:
```bash
python3 fetch-dynamic-content.py https://zolotayabalka.ru/company/sort/ --extract-mode text --wait-for ".sorts-list" --wait-timeout 15000
```

## Технические детали

Навык реализован как гибридное решение:
- **Python скрипт** handles argument parsing and orchestrates the process
- **Node.js Playwright** handles actual browser automation via subprocess
- Этот подход обходит ограничения среды на установку Python пакетов, используя уже доступную системную установку Node.js Playwright

Навык готов к использованию для сбора данных с JavaScript-тяжелых веб-сайтов, SPA приложений и любого контента, требующего взаимодействия для полной загрузки.
