## 1. Описание модели

Разборная модель собаки из 6 уникальных деталей (15 штук в сборке). Все детали соединяются штифтами в пазы с зазором 0.3 мм. Модель не требует поддержек при печати. Собранная длина ~120 мм.

## 4. Сборка

### Порядок сборки

1. **Лапы → Туловище.** Вставить штифты лап в пазы по бокам туловища. 4 лапы — 2 передние, 2 задние. Зазор 0.3 мм обеспечивает плотную посадку без клея.

2. **Голова → Туловище.** Вставить штифт головы в передний паз туловища. Голова должна свободно поворачиваться.

3. **Хвост → Туловище.** Вставить штифт хвоста в задний паз. Хвост фиксируется в одном положении.

4. **Уши → Туловище.** Вставить штифты ушей в верхние пазы. Уши должны держаться, но быть съёмными.

5. **Штифты-фиксаторы.** Использовать для дополнительной фиксации соединений при необходимости (например, если зазор оказался слишком большим из-за погрешности принтера).

### Допуски и зазоры

| Соединение | Номинальный зазор | Примечание |
|-----------|-------------------|------------|
| Лапа-туловище | 0.3 мм | С усилием, без клея |
| Голова-туловище | 0.3 мм | Должна вращаться |
| Хвост-туловище | 0.3 мм | Фиксированное |
| Ухо-туловище | 0.3 мм | Съёмное |

**Важно:** При первом принте рекомендуется напечатать тестовый штифт и паз для проверки зазора. При необходимости скорректировать параметр `clearance` в скрипте.

## 5. Модификация модели

Все параметры модели вынесены в начало файла `make_dog.py`:

```python
body_length = 120.0       # длина туловища
body_width = 35.0         # ширина туловища
body_height = 28.0        # высота туловища
wall_thick = 3.0          # толщина стенок
clearance = 0.3           # зазор для соединений
pin_dia = 4.0             # диаметр штифтов
pin_len = 8.0             # длина штифтов
head_dia = 22.0           # диаметр головы
leg_dia = 8.0             # диаметр лап
leg_len = 25.0            # длина лап
tail_len = 20.0           # длина хвоста
ear_size = 12.0           # размер ушей
joint_dia = 6.0           # диаметр шарниров
```

Для изменения масштаба — умножить все линейные размеры на коэффициент.

## 6. Исходный код

Файл: `make_dog.py`  
Зависимости: `cadquery` (`pip install cadquery`)  
Запуск: `python3 make_dog.py`

Все детали экспортируются в STL с допуском 0.01 мм и угловым допуском 0.1°.

## 7. Ограничения и известные проблемы

- Модель оптимизирована для FDM-печати. Для SLA/SLS зазоры могут потребовать корректировки.
- Тонкие стенки (3 мм) — минимум для надёжной печати на большинстве FDM-принтеров.
- Штифты диаметром 4 мм — минимальный размер для прочного соединения на FDM.
- Рендер превью невозможен без графического окружения (OpenSCAD требует X-сервер).
