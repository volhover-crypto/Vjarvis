## 1. Лабораторный контроль и приёмка

- Обсуждается, что данные по анализам **не подменяются** — есть рефлексия по поводу честности
- Если сильно вылетают из кондиции — не собирают по ГОСТу
- Есть **подрядчик** — сторонняя организация, что усложняет контроль
- Сельхозка — отдельный блок, которому нужны взаимосвязи параметров

**Вывод:** Точка окончания — приёмка. Дальше — задача сельхозки.

## 2. Данные и архитектура

**Нужные данные:**

**Подход:**

**Сертификация:**

## 5. Фитосанитарное состояние

- Контроль для директора СЗ (сельхозземли)
- Пример: высокая трава → комбайн хлопнул семенами → аромат свежескошенного сена в вине
- После инцидента стали отслеживать фитосанитарное состояние
- Другие факторы: гусеницы, улитки, посторонние включения
- **Важно фиксировать** — в итоге вся информация пригодится
- Пример использования: если на участке были улитки — в следующем году планировать химию
